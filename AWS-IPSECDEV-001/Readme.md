# Modulo de PfSense para Terraform
Esse modulo visa a criação de VPN usando Pfsense com Terraform

### Subindo a instancia do Pfsense
No seu terminal digite os seguintes comandos nessa ordem
- Terraform init
- Terraform plan
- Terraform apply
